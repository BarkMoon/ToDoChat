# ToDoChat 安全コマンド許可リスト
# ============================================================================
# ここに載っているコマンドは「実行（都度確認）」モードで自動的に許可されます
# （確認カードは表示されますが、クリック不要）。ファイル・ブランチ・リポジトリの
# 状態を変えない“読み取り専用”の操作だけを載せてください。
#
# 書き方:
#   ・1行1要素。前後の空白は無視されます。
#   ・空行と、# で始まる行はコメントとして無視されます。
#   ・[セクション名] で種類を区切ります。
#
# 迷ったら載せないでください。ここに無いコマンドは従来どおり手動確認になるだけで、
# 安全側に倒れます（＝誤って危険な操作が自動実行されることはありません）。
#
# 注意:
#   ・リダイレクト(> <)、コマンド置換($() ``)、サブシェル、`&`(バックグラウンド)、
#     改行を含むコマンドは、たとえ中身が安全に見えても自動許可されません
#     （この判定は safe_shell.py 側で行っています）。
#   ・`|` `&&` `||` `;` で連結された場合は、連結された全コマンドがこのリストに
#     載っているときだけ自動許可されます（例: `git log | head`）。
# ============================================================================

# [git-listing.*] — 「一覧表示のときだけ」安全な git サブコマンド。
# これらは引数次第で変更操作にもなる（git branch -d / git tag v1 / git remote add

# 自動許可します。位置引数や変更フラグが1つでも混じれば手動確認に回ります。
